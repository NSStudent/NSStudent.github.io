What is Tuist?

Tuist is a command line tool (CLI) that allows you to define, manage, and scale your Xcode projects. This tool doesn't use another environment language like Ruby because is fully implemented in Swift

Install

The first thing that you need to do is install Tuist on your machine with the following command:

curl -Ls https://install.tuist.io | bash

When finished installing the tuist command should work in your terminal.

Commands

Tuist has a collection of subcommands that we will work with on a regular basis. In this first post we try to define the initial subcommand that you need to start with a new project in Tuist.

  • tuist init
  • tuist edit
  • tuist generate
